## Animation Flow

### Onboarding Page Load Sequence

```
1. Header appears        (0ms, immediate)
2. Badge fades in        (300ms delay)
3. Title slides up       (500ms delay)
4. Subtitle appears      (700ms delay)
5. Pills pop in          (900ms delay)
6. Progress bar animates (1100ms delay)
```

### Budget Display Load Sequence

```
Empty State:
1. Badge pulses          (continuous)
2. Title gradient shifts (continuous, 3s cycle)
3. Icon rotates on hover (300ms transition)
4. Button scales on hover(200ms transition)
5. Cards lift on hover   (300ms transition)

Loading State:
1. Shimmer animation     (2s infinite)
2. Pulse effect          (1.5s infinite)
3. Spinner rotation      (1s infinite)
```
